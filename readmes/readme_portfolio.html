August Ollé's portfolio project

1. Objectives

The main objective set for this project was building a well structured online portfolio, covering the next goals:

  • Clear and correct HTML and CSS structure,
  • Neat, balanced, design,
  • Implementing skills using tendencies such as parallax and SVG animation,
  • Giving some contact options via social media, email or a contact form.
  • Easy, ellegant functionality.

2. Design

Four keywords were set as guidelines for the web design: clearness, originality, appeal and comfort. I started with the practical idea of depth and floating elements in a 3d space as an starting point for the design of the site.

Color palette

I picked an analogous harmony palette based on blue tones. Blue is known to convey an idea of stability and reliability. I wanted the site to be colorful and appealing but still easy for the sight.
This palette is kept along the site to mantain concordance. I chose a greyish blue as an accent colour, mainly in buttons.

Typography

Following the initial idea of clearness and comfort, only one font family is used, and it is 'Motserrat', because of its round, wide, traces. They add high readability while looking modern and appealing.

Layout

The page design has a simple layout with a common, neat header, with a logo and a menu icon('burguer') on mobile devices. The menu is visible, fixed on the header, when seen on desktop screens. Tags and text boxes have round corners and are centered to leave free space on both sides on computer screens.

3. Main features

Sections

The site has a main page and the two following sections:

  • Projects, where the user can have an overview of different projects I completed and have access to navigate the project or visit a report (another HTML page) that resumes some of the highlights of the projects and ideas for further versions.
  • Contact, with a contact form where the user can leave a message, that will be sent by email when submitted.

Responsive Design

The web is fully responsive. The design is made to keep the idea of space and depth on all screens. I used media queries at 1000px and 1400px-wide screens, flexbox and grid are used in CSS to reach this goal.

Javascript functionalities

On the main page, there is a parallax efect landscape with some hot air balloons raising to the sky. There is a readable message in words that move in the 3d space created with the parallax effect.
Just under the parallax section, there are two animated SVG, displaying my name and the words 'web development'. They show up as the user scrolls down the page. They emphasize my name and role, while giving some dynamics to the page.
In the Contact page, there is a contact form with an effect on the labels of each input. They are animated using CSS and Javascript.
