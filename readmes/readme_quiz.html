1. Objectives

The main objective set for this project was building a well structured online portfolio, covering the next goals:

  • Clear and correct HTML and CSS structure,
  • Neat, balanced, design,
  • Showing some skills using tendencies such as parallax and SVG animation,
  • Giving some contact options via social media, email or a contact form.
  • Full, easy functionality.

2. Design

Four keywords were set as guidelines for the web design: clearness, originality, appeal and comfort. I started with the practical idea of depth and floating elements in a 3d space as an starting point for the design of the site.

Color palette

I choosed a purple-blue (#6d6dc7) as main color. Blue is known to transmit an idea of stability, reliability, while purple evoques calmness. I wanted the site to be colorful and appealing but still easy for the sight. The gradient on the background helps creating an illusion of depth in the main page, along with the parallax effect.
For the hole site, an analogous color palette is used, to evoque the sense of concordance along the site. I chose just a light blue as an accent colour in buttons.

Typography

Following the initial idea of clearness and comfort, only one font family is used, and it is 'Motserrat', because of its round, wide, traces. They add high readability while looking modern and appealing.

Layout

The page design has a simple layout with a common, neat header, with a logo and a menu icon('burguer') on mobile devices. The menu is visible, fixed on the header, when seen on desktop screens. Tags and text boxes have round corners and are centered, leaving free space on both sides on computer screens.

3. Main features

Sections

The site has a main page and the three following sections:

  • Projects, where the user can have an overview of different projects I completed and have access to navigate the project or visit a report (another HTML page) that resumes some of the highlights of the projects and ideas for further versions.
  • About, with some general information about me and my work.
  • Contact, with a contact form where the user can leave a message, that will be sent by email when submitted.

Responsive Design

The web is fully responsive. The design is made to keep the idea of space and depth on all screens. Media queries, flexbox and grid are used in css to reach this goal.

Parallax

On the main page, there are some 3d icons related to web development floating in 3 different layers. Elements on each layer have a different blur filter and size, so they contribute to the illusion of depth.

Animated SVG

Embedded in the parallax section, there is an animated SVG, displaying my name and the words 'web development'. They show as the user scrolls down the page.

4. Future improvements

Parallax implementation on all sections.

In order to reinforce the sense of continuity in the design along the web. The background where all the web elements float would show some of the 3d icons as in the main page, moving across when scrolling down the page.
